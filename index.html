CONSOLE

Below I am going to tell you about a console screen and steps to download VISUAL STUDIO CODE and how to open CONSOLE SCREEN.

WHAT IS A CONSOLE SCREEN?

An instance of a Windows Console has a screen buffer and an input buffer. It allows console apps to run inside a window or in hardware text mode (so as to occupy the entire screen). ... In 2019, the Windows Console infrastructure was open-sourced under the MIT License, alongside Windows Terminal.

STEPS HOW TO DOWNLOAD "VISUAL STUDIO CODE":

1st: Click on this link= https://code.visualstudio.com/download

2nd: Now choose whichever system you use out of the options provided on the website.

Once you click on download a file will be downloaded in your system.

There you go! Your Visual studio code has been downloaded!

STEPS HOW TO CHECK THE "CONSOLE SCREEN":

1st: Open the web-page of which you want to see the console screen.

2nd: Now do a right click.

Then click on the option which says INSPECT in your drop-down.

Now it will open something on your screen. Which will have an option of CONSOLE. Please click that.

There you go! You just learnt how to check the console screen of a web-page.
